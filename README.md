## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the database

```bash
$ docker compose up -d database
```

## Running the app

```bash
$ pnpm run start:dev
```

## Running the seed

```bash
$ pnpm seed
```

## Test

```bash
# unit tests
$ pnpm run test
```

## Dependencias utilizadas

- nestjs
- bcrypt
- class-transformer
- class-validator
- dotenv
- joi
- mysql2
- passport
- passport-jwt
- passport-local
- reflect-metadata
- rxjs
- swagger-ui-express
- typeorm
